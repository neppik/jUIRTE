jUIRTE

A really simple (and light) Rich Text Editor (wysiwyg) for Jquery UI

Featuring:

  • Buttons & Buttonsets
  • Expandable
  • Cross-browser
  • Very,very,very,very SMALL!

Example

$(document).ready(function(){ 
 $("#myEditor").wysiwyg();
});
